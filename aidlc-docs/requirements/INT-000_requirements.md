# INT-000 Requirements（機能要件）

## FR-001: 備品マスタ管理

### FR-001-1: 備品登録
- 管理者は備品名・説明・返却期限日数（デフォルト）を入力して備品を登録できる
- 登録時に備品IDが自動採番される
- 入力項目: 備品名（必須）、説明（任意）、デフォルト返却期限日数（任意）

### FR-001-2: 備品編集
- 登録済みの備品情報（名前・説明・返却期限日数）を編集できる
- 備品IDは変更不可

### FR-001-3: 備品一覧表示
- 登録済み備品をカード形式またはテーブル形式で一覧表示する
- 各備品の貸出中/返却済みステータスを表示する

### FR-001-4: QRコード生成・表示
- 備品ごとにQRコードを生成・表示できる
- QRコードには備品IDをエンコードする
- 印刷用QRコード表示（印刷ボタン）を提供する

---

## FR-002: QRコードスキャンによる貸出・返却

### FR-002-1: QRコードスキャン画面
- スマホブラウザからカメラを起動し、QRコードをスキャンできる
- スキャン成功後、備品情報と現在のステータスを表示する

### FR-002-2: 貸出処理
- 返却済み状態の備品QRをスキャンした場合、貸出者名と返却予定日を入力して貸出を記録できる
- 貸出記録には: 備品ID・貸出者名・貸出日時・返却予定日を保存する

### FR-002-3: 返却処理
- 貸出中状態の備品QRをスキャンした場合、ワンタップで返却を記録できる
- 返却記録には: 返却日時を保存する

---

## FR-003: 貸出状況ダッシュボード

### FR-003-1: 現在の貸出一覧
- 現在貸出中の備品を一覧で表示する
- 表示項目: 備品名・貸出者名・貸出日・返却予定日・経過日数

### FR-003-2: 返却期限アラート
- 返却予定日を超過している貸出に対してアラート（赤色など）を表示する
- 返却予定日当日・翌日の貸出に対して警告（黄色など）を表示する

---

## FR-004: 貸出・返却履歴

### FR-004-1: 履歴一覧
- すべての貸出・返却の履歴を時系列（新しい順）で表示する
- 表示項目: 備品名・貸出者名・貸出日時・返却日時・ステータス

### FR-004-2: 備品別履歴
- 特定の備品に絞った貸出・返却履歴を表示できる

---

## データモデル（概要）

### equipments（備品マスタ）
| カラム | 型 | 説明 |
|---|---|---|
| id | serial PK | 備品ID |
| name | text | 備品名 |
| description | text | 説明 |
| default_return_days | integer | デフォルト返却期限日数 |
| created_at | timestamp | 登録日時 |

### loans（貸出記録）
| カラム | 型 | 説明 |
|---|---|---|
| id | serial PK | 貸出記録ID |
| equipment_id | integer FK | 備品ID |
| borrower_name | text | 貸出者名 |
| loaned_at | timestamp | 貸出日時 |
| due_date | date | 返却予定日 |
| returned_at | timestamp | 返却日時（NULLなら貸出中） |
