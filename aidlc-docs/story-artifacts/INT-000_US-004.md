# INT-000 US-004: 貸出状況ダッシュボード・アラート

## As a（誰として）

管理者として

## I want（何をしたいか）

現在貸出中の備品一覧と返却期限アラートをダッシュボードで確認したい

## So that（なぜ・何のために）

誰が何を借りているかを一目で把握し、返却期限切れを早期に検知して対処できるようにするため

## Acceptance Criteria（受け入れ基準）

- [ ] ダッシュボード（`/dashboard` または `/`）に貸出中備品の一覧が表示される
- [ ] 一覧に表示される項目: 備品名・貸出者名・貸出日・返却予定日・経過日数
- [ ] 返却期限を超過した貸出は赤色ハイライト（アラート）で表示される
- [ ] 返却予定日が当日または翌日の貸出は黄色ハイライト（警告）で表示される
- [ ] 貸出中件数のサマリー（例: 「現在3件貸出中 / うち1件期限超過」）が表示される
- [ ] ページをリロードすると最新の貸出状況が反映される

## 関連要件

- FR-003-1: 現在の貸出一覧
- FR-003-2: 返却期限アラート

- `/` または `/dashboard` - 貸出状況ダッシュボード
