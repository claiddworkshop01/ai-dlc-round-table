# INT-000 US-005: 貸出・返却履歴閲覧

## As a（誰として）

管理者として

## I want（何をしたいか）

備品の貸出・返却の全履歴を時系列で閲覧したい

## So that（なぜ・何のために）

過去の利用状況を把握し、備品の使用頻度や問題のある利用パターンを確認できるようにするため

## Acceptance Criteria（受け入れ基準）

- [ ] 履歴ページ（`/history`）に全貸出・返却記録が新しい順で一覧表示される
- [ ] 一覧に表示される項目: 備品名・貸出者名・貸出日時・返却日時・ステータス（返却済み/貸出中）
- [ ] 備品一覧ページの各備品から「履歴を見る」リンクで、その備品の履歴に絞り込んで表示できる
- [ ] 貸出中のレコードは返却日時が「-」（未返却）と表示される

## 関連要件

- FR-004-1: 履歴一覧
- FR-004-2: 備品別履歴

## 想定画面

- `/history` - 全履歴一覧
- `/equipments/[id]/history` - 備品別履歴
