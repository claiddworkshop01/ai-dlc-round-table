# INT-000 US-001: 備品マスタ登録・編集

## As a（誰として）

管理者として

## I want（何をしたいか）

備品の名前・説明・デフォルト返却期限日数を登録・編集したい

## So that（なぜ・何のために）

管理対象の備品情報を正確に維持し、貸出管理の基盤を整えるため

## Acceptance Criteria（受け入れ基準）

- [ ] 備品一覧ページが存在し、登録済み備品がカード/テーブルで表示される
- [ ] 「新規登録」ボタンから備品名・説明・デフォルト返却期限日数を入力できる
- [ ] 備品名は必須項目で、空の場合はバリデーションエラーが表示される
- [ ] 登録後、一覧に新しい備品が即時反映される
- [ ] 各備品の「編集」ボタンから情報を変更・保存できる
- [ ] 編集後、一覧に変更内容が即時反映される

## 関連要件

- FR-001-1: 備品登録
- FR-001-2: 備品編集
- FR-001-3: 備品一覧表示

## 想定画面

- `/equipments` - 備品一覧ページ
- `/equipments/new` - 備品登録フォーム
- `/equipments/[id]/edit` - 備品編集フォーム
