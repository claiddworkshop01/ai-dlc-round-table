# INT-000 US-002: QRコード生成・表示・印刷

## As a（誰として）

管理者として

## I want（何をしたいか）

備品ごとのQRコードをブラウザ上で生成・表示し、印刷したい

## So that（なぜ・何のために）

備品にQRコードシールを貼ることで、利用者がスマホで読み取って貸出・返却を行えるようにするため

## Acceptance Criteria（受け入れ基準）

- [ ] 備品一覧または備品詳細ページから「QRコード表示」ボタンを押せる
- [ ] 備品IDをエンコードしたQRコードが画面に表示される
- [ ] QRコードの下に備品名が表示される
- [ ] 「印刷」ボタンを押すとQRコードのみ印刷プレビューが開く
- [ ] 生成されたQRコードをスマホカメラで読み取ると、正しい備品IDが取得できる

## 関連要件

- FR-001-4: QRコード生成・表示

## 想定画面

- `/equipments/[id]` - 備品詳細ページ（QRコード表示含む）
- または `/equipments/[id]/qr` - QRコード専用表示ページ

## 技術メモ

- QRコード生成: `qrcode` npmパッケージまたは `react-qr-code` を使用
- エンコード内容: 備品ID（例: `equipment:123`）またはURLパス（例: `/scan?id=123`）
