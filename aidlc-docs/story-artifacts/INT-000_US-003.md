# INT-000 US-003: QRスキャンによる貸出・返却

## As a（誰として）

備品を借りたい（または返却したい）利用者として

## I want（何をしたいか）

スマホのブラウザでQRコードをスキャンし、貸出または返却を記録したい

## So that（なぜ・何のために）

備品の受け渡しを素早く・正確に記録でき、管理台帳への手書きや管理者への申告が不要になるため

## Acceptance Criteria（受け入れ基準）

- [ ] スキャン用ページ（`/scan`）にアクセスすると、カメラが起動してQRコードを読み取れる
- [ ] QRスキャン成功後、備品名と現在のステータス（貸出中/返却済み）が表示される
- [ ] 備品が「返却済み」状態の場合:
  - [ ] 貸出者名と返却予定日を入力するフォームが表示される
  - [ ] 「貸出する」ボタンで貸出が記録される
- [ ] 備品が「貸出中」状態の場合:
  - [ ] 貸出中の情報（貸出者名・貸出日・返却予定日）が表示される
  - [ ] 「返却する」ボタンで返却が記録される
- [ ] 貸出・返却後、完了メッセージが表示される

## 関連要件

- FR-002-1: QRコードスキャン画面
- FR-002-2: 貸出処理
- FR-002-3: 返却処理

## 想定画面

- `/scan` - QRスキャン画面（カメラ起動）
- スキャン後、同画面内でモーダルまたはインライン表示で貸出・返却フォームを表示

## 技術メモ

- カメラアクセス: ブラウザの `MediaDevices.getUserMedia()` API または `react-qr-reader` 等のライブラリ
- スマホ対応必須（iOS Safari / Android Chrome）
